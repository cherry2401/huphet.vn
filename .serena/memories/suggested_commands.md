Project commands:
- npm run dev : start Next.js dev server
- npm run build : build for production
- npm run start : run production server
- npm run lint : run ESLint
- npm run test : run Vitest suite
- npm run extract:microsite : extract Shopee microsite data
- npm run sync:shopee-cache -- <slug> : sync Shopee cache
- npm run sync:partner-cache -- <slug> : sync partner cache
- npm run worker:browser-sync -- <slug> : run browser sync worker
- npm run worker:bootstrap : bootstrap worker state
- npm run worker:bootstrap-live : bootstrap live worker state
- npm run worker:login : helper for browser login
- npm run worker:affiliate-login : helper for affiliate login
- npm run worker:affiliate-open-profile : open affiliate profile
- npm run worker:install-browser : install Playwright Chromium
Useful shell commands in this environment use Unix shell syntax via bash on Windows, e.g. ls, pwd, git, and quoted absolute paths like /i/Website/hup-het/shopee-aff-site.
