Shopee Aff Site is a Next.js 16 affiliate/marketing website for Shopee deals, live content, flash sale, and voucher pages, with internal sync/status APIs, admin tooling, Supabase-backed cache storage, and Playwright-based worker scripts for VPS/browser-assisted data syncing. Main code areas are src/app for routes/UI, src/lib for domain logic and adapters, middleware.ts for request filtering, docs for architecture/env guidance, scripts for sync utilities, and worker for browser automation.
